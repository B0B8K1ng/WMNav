WMNav: Empowering Object Goal Navigation via Using VLM as World Models

Abstract

Object Goal Navigation--requiring an agent to locate a specific object in an unseen environment--remains a core challenge in embodied AI.

Although recent progress in Vision-Language Model (VLM)--based agents have demonstrated promising perception and decision-making through prompting, none has yet established a fully modular world-model design. We propose WMNav, a framework mainly consists of five distinct modules--perception, planning, cost estimation, memory management, and action. By decomposing according to a human-like thinking process, WMNav effectively alleviates model hallucination by making decisions based on the feedback difference between word model plan and observation.

The system subsequently evaluates task costs and executes actions until the object is found. To further boost efficiency, we incorporate a dynamic Exploration Value Map and implement a two-phase action strategy: broad exploration followed by precise localization. On HM3D, WMNav surpasses state-of-the-art methods in both success rate and exploration efficiency. WMNav reaches SR 58.1%(+3.2% to SOTA) and SPL 31.1%(+6.8% to SOTA).

Method Overview

The WMNav framework. After acquiring RGB-D images and pose information from the environment, WMNav primarily involves three processes. First, it predicts the probability of the goal appearing in different directions within the panoramic image, while simultaneously updating the Exploration Value Map in memory. Then, based on the map, the scores of each direction are projected back onto the panoramic image, and the direction with the highest score is selected. Second, the new subtask is determined based on the selected image and the last subtask feedback. Third, the subtask and the image with the action space constructed by the Action proposer and Goal proposer are used to determine the final polar coordinate vector action for execution.
